# 소개

동행복권 사이트내에 계정에 예치금만 넣어두시면 이후 매주 로또와 연금복권을 구입하고 당첨을 체크하여 알려드려요!

## 🎰 로또 구매 전략

### AUTO 모드
- 완전 자동 선택
- 서버가 무작위로 번호 선택

### SUPERSTITION 모드 (미신 전략) ⭐ NEW!
**옵션 C: 혼합 전략**
- ❌ **내가 지난주에 구매한 번호는 제외**
  - "내가 산 번호는 이번주에 안 나온다" 미신 적용
- ✅ **지난주 당첨번호 중 하나씩 고정**
  - "당첨번호 중 일부가 다시 나온다" 미신 적용
  - A게임: 당첨번호 1번째 고정 + 랜덤 5개
  - B게임: 당첨번호 2번째 고정 + 랜덤 5개
  - C게임: 당첨번호 3번째 고정 + 랜덤 5개
  - (계속...)
- 🎲 나머지는 랜덤 선택

# 사용법

![](./.github/images/check.png)

- 레포지토리를 `fork` 합니다.
- Settings - Secrets - Add a new secret 메뉴로 들어갑니다
- 환경 변수들을 만들어 줍니다 (.env.sample 참조)
  - `LOTTO_STRATEGY`: `AUTO` 또는 `SUPERSTITION` 선택
- 매주 로또 및 연금복권 구매 및 당첨 과정을 자동으로 알려드려요 🎉
